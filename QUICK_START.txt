└─────────────────────────────────────────────────────────────────────────────┘

  Variables de entorno (.env):
  ├── FLASK_PORT=5000          (Puerto del servidor)
  ├── FLASK_DEBUG=false        (Debug mode)
  ├── FLASK_ENV=production     (Entorno)
  └── CORS_ORIGINS=*           (Orígenes CORS - en desarrollo)

  Dependencias a instalar:
  ├── pip install -r requirements.txt
  └── pip install flask-cors

┌─────────────────────────────────────────────────────────────────────────────┐
│ 🔧 TROUBLESHOOTING                                                          │
└─────────────────────────────────────────────────────────────────────────────┘

  ❌ "FFmpeg no encontrado"
     → Windows: descarga ffmpeg.org
     → Linux: sudo apt install ffmpeg
     → Mac: brew install ffmpeg

  ❌ "No se conecta al servidor"
     → ¿Está corriendo? python src/app_cors.py
     → URL correcta: http://localhost:5000 (sin slash final)
     → ¿Puerto correcto? (default 5000)

  ❌ "Error de CORS"
     → Usa app_cors.py, no app.py
     → CORS ya está configurado para *

  ❌ "Chunks no se limpian"
     → Se eliminan automáticamente al terminar
     → Revisa permisos en audios/uploads/
